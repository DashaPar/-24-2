# ИНСТРУКЦИЯ ПО ЗАПУСКУ

## Требования
- Java 11+
- Maven 3.6+

## Запуск
1. Открыть командную строку в папке проекта
2. Выполнить: `mvn clean compile`
3. Выполнить: `mvn exec:java -Dexec.mainClass="com.workscheduler.MainApp"`

## Первый вход
- Логин: `admin`
- Пароль: `admin123`

## Управление
- Добавить задачу → кнопка "Добавить"
- Редактировать → выбрать задачу → "Редактировать"
- Удалить → выбрать задачу → "Удалить"
- Отчёты → меню "Отчёты"
- Журнал действий → меню "Отчёты" → "Журнал действий"
- Диаграмма Ганта → выбрать вид "Диаграмма Ганта"

## Папки
- `reports/` - сюда сохраняются отчёты (создаётся автоматически)
- `work_scheduler.db` - файл базы данных (создаётся автоматически)

## Ошибки
- Если не запускается → проверьте Java: `java -version`
- Если ошибка компиляции → `mvn clean` затем `mvn compile`
- Если проблемы с БД → удалите `work_scheduler.db` и перезапустите
